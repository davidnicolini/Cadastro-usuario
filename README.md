# cadastro-usuario
Sistema de cadastro de usuário com Spring boot

### 🔧 Tecnologias usadas:
Java 23 (JDK 23)
Spring Boot
H2 Database
Spring Data JPA
Lombok
Maven

📚 Objetivo aprender a:
 * Como estruturar um projeto com Spring Boot
 * O que é um CRUD na prática (Create, Read, Update, Delete)
 * Como configurar banco de dados H2

💥 Esse projeto está em andamento para melhorias.
